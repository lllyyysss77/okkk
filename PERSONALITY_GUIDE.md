# 🎭 AI 身份预设指南

这个语音助手支持 6 种不同的 AI 身份预设，可以快速切换 AI 的行为风格。

## 可用的身份预设

### 1. 📌 Default（默认）
**描述**: 通用型助手，快速高效地回答问题
- **全球版**: 标准语音助手，保持回答简洁
- **中国版**: 针对中国大陆用户的流畅服务

### 2. 📚 Academic（学术）
**描述**: 专业学术顾问，提供深度研究型回答
- 使用正式的学术语言
- 提供引用和参考资料
- 适合：研究、论文写作、学术讨论

### 3. 💼 Sales（销售）
**描述**: 热情的销售顾问，帮助客户找到解决方案
- 既有说服力又要诚实透明
- 注重客户需求分析
- 适合：产品推介、客户咨询、销售支持

### 4. 👨‍🏫 Teacher（教师）
**描述**: 耐心的教育工作者，用简单方式解释复杂概念
- 多用例子和比喻
- 循序渐进的讲解
- 适合：知识学习、问题解答、教育支持

### 5. 😊 Friendly（友好）
**描述**: 友好的聊天伙伴，轻松有趣的对话风格
- 使用口语和表情符号
- 保持对话轻松
- 适合：闲聊、陪伴、娱乐对话

### 6. 🎯 Professional（专业）
**描述**: 商业顾问，提供实用可行的建议
- 简洁明了的回答
- 数据驱动的分析
- 适合：商业建议、项目管理、战略规划

## 如何使用

### 方法 1：通过 UI 快速切换（推荐）

1. **打开语音助手** → 点击右上角的 ⚙️ 设置图标
2. **在 "AI Personality" 部分选择身份**
3. **现有的系统指令会自动更新**
4. **断开连接后重新连接以应用新身份**

### 方法 2：自定义指令

如果预设不满足你的需求，可以在 "Custom Instruction" 框中手动编写系统提示：

```
你是一个专业的...
请以...的风格回答
...
```

## 示例对话

### 同一个问题，不同身份的回答

**问题**: "如何写好一篇文章？"

**📌 Default**: "写好文章的关键是：清晰的结构、有力的论点和简洁的表达。"

**📚 Academic**: "文章写作涉及多个维度：(1) 组织结构 - 引言、主体、结论; (2) 论证方法 - 演绎法、归纳法等; (3) 修辞手法 - 排比、对偶等。参考 Bloom 分类法..."

**💼 Sales**: "好文章是一个强大的销售工具！让我帮你：明确目标读者、突出核心价值、用故事吸引注意力..."

**👨‍🏫 Teacher**: "把文章比作一个三明治：顶层是开头（吸引注意），中间层是内容（充满营养），底层是结尾（留下印象）。让我一步步教你..."

**😊 Friendly**: "哈哈，写文章嘛！其实就是把脑子里的想法整理出来 😄 多看好文章、多练习、不要太纠结完美..."

**🎯 Professional**: "高效写作流程：(1) 收集数据和案例 (2) 建立框架 (3) 快速初稿 (4) 反复修改 (5) 最终校对。预计时间：2-4小时。"

## 代码配置

身份预设定义在 [types.ts](types.ts) 中的 `PERSONALITY_PRESETS` 对象。

要添加新的身份，只需编辑该对象：

```typescript
export const PERSONALITY_PRESETS = {
  GLOBAL: {
    YOUR_NEW_PERSONALITY: "你的系统指令..."
  },
  CHINA_MAINLAND: {
    YOUR_NEW_PERSONALITY: "你的系统指令（中文）..."
  }
}
```

然后在 `PersonalityType` 枚举中添加对应的值即可。

## 💡 最佳实践

1. **选择合适的身份** - 根据使用场景选择身份
2. **如需微调** - 在自定义框中修改具体内容
3. **保持一致性** - 同一对话中不要频繁切换身份
4. **测试回答** - 同一问题尝试不同身份，看哪个效果最好
